# react-python-app

## プロジェクトの始め方
- 下記コマンド実行
>docker-compose build

>docker-compose up -d

## reactプロジェクト新規作成
- 下記コマンド実行
>docker-compose run --rm node sh -c "npm install -g create-react-app && create-react-app reactapp"

## python(Dijngo)新規作成
- 下記コマンド実行
>docker-compose run backend django-admin.py startproject python .
